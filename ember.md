# Mac 编译环境准备

在venv里设定制定头文件路径：

```shell
source /Users/yunkun/pro/ceasphere/ember-csi/venv/bin/activate
export C_INCLUDE_PATH=/Library/Developer/CommandLineTools/Library/Frameworks/Python3.framework/Versions/3.8/Headers
pip install cinderlib
```
